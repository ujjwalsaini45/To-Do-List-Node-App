🚀 Full-Stack To-Do List SPA | React & Node.js
📌 Overview

Developed a Full-Stack To-Do List Single Page Application (SPA) using React and Node.js.
The application enables users to efficiently manage daily tasks with a smooth and responsive user experience.

✨ Features
Add, delete, and view todos
Smooth navigation with SPA architecture
Responsive and user-friendly UI
Persistent data using Local Storage
⚙️ Tech Stack
Frontend: React (useState, useEffect, Props), React Router
Backend: Node.js
Storage: Local Storage
Tools: Git, GitHub, VS Code
💡 Key Highlights
Implemented modern React practices using Hooks
Designed a scalable Single Page Application architecture
Efficient state management for dynamic UI updates
Clean and modular component-based structure
🎯 Learning Outcome

This project strengthened my understanding of full-stack development, React fundamentals, and building interactive, real-world applications.

🔖 Skills Demonstrated

React • JavaScript • SPA Development • Node.js • State Management • UI Development
